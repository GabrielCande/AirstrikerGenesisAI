# Airstriker-Genesis AI

Repository with implemente NEAT, PPO and Genetic agents for the game Airstriker-Genesis

# Dependencies
## To run the code you must have the following libraries installed:
`gym-retro == 0.8.0`

`neat-python == 0.92`

`numpy >= 1.24.4`

`pygame == 2.5.2`

`shimmy == 0.2.1`

`stable_baselines3 == 2.3.2`
